# ExampleWidgets
Simple examples of different widgets for Android written in Kotlin
List of widgets used:
- TextView
- EditText
- Button
- RadioButton
- CheckBox
- RatingBar
- Spinner
- Switch
- ProgressBar
- AutoComplete TextView
- TextSwitcher
- Checked TextView
- ToggleButton
- Custon CheckBox
- Horizontal ScrollView
- Horizontal, Vertical, Rotated Lines and Underline using View Tag
